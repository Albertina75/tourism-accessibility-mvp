# Turismo Accesibilidad MVP

## Descripción

Este proyecto es una aplicación mínima viable para un sistema de accesibilidad en turismo. Utiliza Express para el servidor backend y React para la interfaz de usuario.

## Requisitos

- [Node.js](https://nodejs.org/) (versión 14 o superior recomendada)
- [npm](https://www.npmjs.com/) (viene con Node.js)

## Instalación

1. **Clona el repositorio**

   ```bash
   git clone https://github.com/Albertina75/tourism-accessibility-mvp.git

2. **Navega al directorio del proyecto**
cd tourism-accessibility-mvp

3. **Instala dependencias**
cd tourism-accessibility-mvp
